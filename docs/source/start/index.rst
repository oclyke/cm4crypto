Getting Started
===============

How to get up and running with the cm4crypto adaptor

Prerequisites
*************

You will need:

* Raspberry Pi Compute Module 4
   * you may choose the appropriate model for your needs depending on size of RAM, 
     type/amount of storage, and WiFi support
* cm4crypto adapter kit
* power supply
   * USB-C cable or PoE
* firmware flashing method
   * SD card *OR* USB-C cable for modules with EMMC storage
